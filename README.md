# MySQL Self-Training 

A personal **MySQL self-training repository** designed to improve database skills through exercises, queries, and small practice projects.  

---

##  Overview

This repository contains **notes, SQL exercises, and practice databases** covering MySQL fundamentals and intermediate topics. It’s intended for self-learners who want a hands-on approach to mastering relational databases.

---

##  Topics Covered

- Database Basics: creating databases, tables, and relationships  
- Data Types and Constraints  
- CRUD Operations (Create, Read, Update, Delete)  
- Joins: INNER, LEFT, RIGHT, FULL  
- Aggregate Functions and Grouping  
- Subqueries and Nested Queries  
- Indexing and Keys  
- Views and Stored Procedures  
- Triggers  
- Transactions and ACID Principles  
- Practice Projects with sample datasets  

---

## 🛠️ Tech Stack

- **Database:** MySQL  
- **Tools:** MySQL Workbench, phpMyAdmin,
